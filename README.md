# cuota
プログラム初学者を対象に、情報の共有・検索をテーマにしたサイト
# 概要
会員ユーザー
・記事の投稿
・他ユーザーの記事に役に立った！ボタンで評価
・記事のキープ
・マイページで、自分が投稿した記事とキープした記事を確認できる
非会員ユーザー
・記事の閲覧のみ可能
管理者ユーザー
・全記事の閲覧
・投稿者と日付で記事の絞り込みが可能
